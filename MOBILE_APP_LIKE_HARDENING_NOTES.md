# Mobile App-Like Hardening

Umgesetzt:

- Backoffice bekommt eigene mobile Bottom-Navigation.
- Backoffice Sidebar wird zu einem echten App-Drawer.
- Backoffice Topbar wird zweizeilig mit Menü, Kontext und Kundensuche.
- Admin-Karten, Tabellen, Tabs, Buttons und Forms werden mobil app-like optimiert.
- Tabellen sind mobil horizontal scrollfähig.
- Frontoffice Bottom-Nav/Safe-Areas/Touch-Ziele wurden weiter verfeinert.
- Safe-Area-Unterstützung für iPhone Dynamic Island/Home Indicator.
- PWA/Standalone-Polish ergänzt.
- Drawer sperrt den Body-Scroll, damit die App sich nativer anfühlt.

Keine neue Supabase-Migration nötig.
