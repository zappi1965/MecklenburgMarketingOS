# Customer Mail & Package Live Binding

Dieses Paket verbindet vorbereitete Kundenregistrierung, Einladung, Mails und Paket-Freigaben stärker miteinander.

## Mail-Domain

Standard:

- Firma: `MecklenburgMarketing GbR`
- Domain: `mecklenburgmarketing.de`
- Absender: `MecklenburgMarketing GbR <noreply@mecklenburgmarketing.de>`
- Reply-To: `info@mecklenburgmarketing.de`

## Ablauf Registrierung

1. Kunde registriert sich öffentlich.
2. System legt pending customer / registration / profile an.
3. Kunde erhält Eingangsbestätigung.
4. Admin erhält Hinweis.
5. Admin gibt Registrierung frei.
6. Kunde erhält Freischaltungs-Mail.
7. Pakettools werden automatisch freigegeben.

## Ablauf Einladung

1. Admin erstellt Einladung.
2. Kunde erhält Einladungs-Mail.
3. Kunde setzt Passwort und nimmt Einladung an.
4. Kunde wird aktiv.
5. Pakettools werden automatisch freigegeben.
6. Admin erhält Aktivierungs-Hinweis.
