# V46 Value Dashboard + Report Fullbuild

## Ziel

V46 macht MMOS optisch hochwertiger und verkaufsfaehiger, indem bestehende Tools zu einer klaren Kundennutzen-Ansicht zusammengefuehrt werden.

## Neue Seiten

- `/value-dashboard`
- `/reports/value/[id]`

## Verknuepfte Datenquellen

Das Value Dashboard nutzt:

- Kunden
- SEO Snapshots
- QR-Kampagnen
- Review Feedback
- Loyalty Kunden
- Loyalty Transaktionen
- Loyalty Rewards
- Tickets
- Termine
- Warteliste
- Inbox-Nachrichten
- Listings / NAP Score
- Rechnungen
- Zahlungslinks
- Gutscheine
- Referral Kampagnen
- Referral Events
- Leads

## Funktionen

- Kundenauswahl
- Value Score 0-100
- Potenzial-Indikator
- QR-Scans und Conversions
- Review- und Feedback-Zahlen
- SEO-Sichtbarkeit
- Listing/NAP-Qualitaet
- Loyalty-Status
- offene Inbox / Tickets
- Payments & Gutscheine
- Empfehlungen
- naechste Admin-Aktionen
- Monatsreport erstellen
- Report-Historie
- druckbare Reportseite

## Demo-Daten

V46 nutzt V45-Demo-Daten automatisch mit. Zusaetzlich legt die Supabase-Migration einen Demo-Value-Report an.
