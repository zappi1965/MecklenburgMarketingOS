
# Responsive QA Checklist

Breakpoints:
- 1440px Desktop
- 1280px Laptop
- 1024px Tablet Landscape
- 768px Tablet Portrait
- 430px iPhone Pro Max
- 390px iPhone
- 360px Android Small

Prüfen:
- Sidebar Verhalten
- Tabellen horizontal scrollen
- Charts nicht abgeschnitten
- Buttons erreichbar
- Modals/Dropdowns sichtbar
- Header bricht sauber um
- Formulare vollständig bedienbar
- Touch-Ziele groß genug
