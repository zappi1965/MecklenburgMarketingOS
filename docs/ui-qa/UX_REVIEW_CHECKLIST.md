
# UX Review Checklist

- Jeder Hauptbutton hat klares Feedback
- Jede leere Liste hat Empty State
- Jeder Ladezustand hat Skeleton/Spinner
- Kritische Aktionen haben Confirm
- Fehler sind verständlich
- Tabellen haben klare Spaltennamen
- Charts haben sinnvolle Labels
- Admin/Kundenkontext ist sichtbar
- Paketstatus ist klar erkennbar
- Export/PDF-Aktionen sind auffindbar
