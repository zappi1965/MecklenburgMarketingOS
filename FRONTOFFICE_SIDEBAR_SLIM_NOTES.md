# Frontoffice Sidebar Slim

Umgesetzt:

- Frontoffice-Sidebar schlanker gestaltet
- Tools nicht entfernt, sondern sinnvoller zusammengefasst
- Navigation als Accordion umgesetzt
- Pro Bereich ist standardmäßig nur eine Gruppe geöffnet
- Aktive Tool-Gruppe öffnet sich automatisch
- Admin-Frontoffice von 7 auf 5 Gruppen verdichtet
- Kunden-Frontoffice von 6 auf 4 Gruppen verdichtet

Ziel:
Weniger visuelle Überladung, klarere Orientierung, trotzdem voller Funktionsumfang.
